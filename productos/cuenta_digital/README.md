# Cuenta Digital

Análisis de apertura, seguimiento y monetización de cuentas digitales.
Incluye segmentación por generaciones, campañas RTM y validación con Tableau.

## Estructura dentro de cada mes

```
2026-XX/
├── queries/      → Archivos .sql con las consultas a SQL Server
├── analysis/     → Scripts Python de análisis con pandas
├── dashboards/   → Apps Dash/Plotly para visualización
└── exports/      → Archivos Excel/CSV generados (no se suben a git)
```
