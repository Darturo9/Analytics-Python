# App Empresarial

Análisis de la aplicación bancaria empresarial. Seguimiento de campañas,
transacciones, conversiones y uso de la app por clientes empresariales.

## Carpetas por mes

| Mes       | Descripción                        |
|-----------|------------------------------------|
| 2026-01   | Enero 2026                         |
| 2026-02   | Febrero 2026                       |
| 2026-03   | Marzo 2026                         |

## Estructura dentro de cada mes

```
2026-XX/
├── queries/      → Archivos .sql con las consultas a SQL Server
├── analysis/     → Scripts Python de análisis con pandas
├── dashboards/   → Apps Dash/Plotly para visualización
└── exports/      → Archivos Excel/CSV generados (no se suben a git)
```
