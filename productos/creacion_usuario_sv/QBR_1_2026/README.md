# creacion_usuario_sv / QBR_1_2026

Dashboard y consultas para el primer trimestre de 2026 (enero-marzo).

## Archivos principales

- `dashboards/dashboard_q1_2026.py`: dashboard de Creacion de Usuario SV enfocado en Q1 2026.
- `sql/conversion_q1_2026.sql`: base de conversion de usuarios para Q1 2026.
- `sql/comunicacionesRTM_q1_2026.sql`: base de comunicaciones RTM para Q1 2026.

## Rango del trimestre

- Inicio: `2026-01-01 00:00:00`
- Fin (exclusivo): `2026-04-01 00:00:00`

## Como ejecutar

```bash
python3 productos/creacion_usuario_sv/QBR_1_2026/dashboards/dashboard_q1_2026.py
```

El dashboard se levanta en:

- `http://127.0.0.1:8054`
